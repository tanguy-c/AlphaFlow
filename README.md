# [AlphaFlow](https://tanguy-c.github.io/AlphaFlow/)

AlphaFlow helps turn text into clear, step-by-step machine instructions.

Instead of checking code charts by hand every time, you can use AlphaFlow to:

- write the text you want to create
- choose the machine configuration that matches your device
- follow each code and mode change in order
- search the full code reference for your machine

Use the app here: https://tanguy-c.github.io/AlphaFlow/

## Share Your Machine Config

If you already have a working machine configuration, please share it so it can be added for everyone.

The best process is:

1. Open AlphaFlow and export your machine config from the `Settings` page.
2. Go to the GitHub issues page: https://github.com/tanguy-c/AlphaFlow/issues
3. Create a new issue and choose `Share a machine config`.
4. Attach your exported config file.
5. Add the machine brand, model, and any useful notes or screenshots of the original code guide.

I will review shared configs and add the useful ones to the project so the whole community can benefit.
